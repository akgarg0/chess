Chess Gaming Server

Chess Gaming server is a gaming server helps two players to play chess with each other anywhere across the world.

Online Since - 24 Jan 2021

Would you like to have a game of CHESS

Application: https://ak-chess.herokuapp.com/
Method: POST
Starting Game ? Put "START" in body
Moving my Queen : Refer blog for notations and rules
https://www.ichess.net/blog/chess-notation/
https://www.chesscoachonline.com/chess-articles/chess-rules
Note:
1. First Request to Application may take some time
2. All the requests sent on 'POST' method should have body as 'plain text
